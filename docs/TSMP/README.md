# TSMP 服务器

TSMP 服务器是原 TrystageSMP,为了便于管理,合并入TrystageServer,并进行了一些更改  
现可以在TrystageServer里使用/smp进入TSMP服务器  


## 服务器功能

### 可用功能
- 服务器内[地图](https://map.tsmp.top)
- 聊天发送图片
- 公共网盘

### 已删除的功能
- 服务器内聊天(chat.tsmp.top)
- 服务器小游戏(如起床战争,空岛战争)
- 服务器主大厅(现使用TrystageServer大厅)
- 基岩版支持(仍在维护)
- TSMP Doc(现使用TrystageServer Wiki)

### 新增功能
- 与TrystageServer同步的Rank系统
- 与TrystageServer同步的Tab样式
