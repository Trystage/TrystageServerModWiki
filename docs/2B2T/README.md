## 服务器特色

### 更快的发展节奏
我们没有长达数年的历史积淀，因此大幅缩短了玩家从0到1的资源积累过程。通过以下方式，你可以迅速武装自己：

- **简化配方**：末影水晶、附魔之瓶现在只需基础材料即可合成。
- **强力工具**：新增多种挖掘、战斗特化装备，助你高效获取资源。
